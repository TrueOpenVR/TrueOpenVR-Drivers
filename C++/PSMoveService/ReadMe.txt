En:
Requires PSMoveService and configuration.
For a 32-bit operating system, copy the contents of the "..\TrueOpenVR\Additionally\PSMoveService\x86" folder to the "C:\Windows" folder.
For a 64-bit operating system, copy the contents of the "..\TrueOpenVR\Additionally\PSMoveService\x64" folder to the "C:\Windows" folder.

Buttons:
The stick is emulated by buttons.
The PS button on both controllers centring rotation the controllers.

More - https://github.com/r57zone/PSMoveService-receive-position/blob/master/README.md

Ru:
Необходим PSMoveService и настройка. 
Для 32 битной операционной системы скопируйте содержимое папки "..\TrueOpenVR\Additionally\PSMoveService\x86" в папку "C:\Windows".
Для 64 битной операционной системы скопируйте содержимое папки "..\TrueOpenVR\Additionally\PSMoveService\x64" в папку "C:\Windows".

Кнопки:
Стик эмулируется кнопками.
PS кнопка на обоих контроллерах центрирует вращение контроллеров.

Подробнее - https://github.com/r57zone/PSMoveService-receive-position/blob/master/README.md
